# Посмотри в окно

## Описание

Верстка страницы сайта с обработкой различных состояний UI-элементов.

## Этапы реализации

1. Верстка основного лейаута
2. Стили текста
3. Стили форм
4. Стили кнопок
5. Состояния карточек
6. Спиннеры, дополнительные состояния
7. Доработка под pixelperfect
8. Доработка по замечаниям ревьюера
